# 🚀 dev-setup

> A simple script to quickly set up a developer environment on Linux.

---

## 📦 Installation

Make sure you have **git** installed.

Then clone the repository:

```bash
git clone https://github.com/ERREKID/dev-setup.git
cd dev-setup
``` 

---

## ⚡ Usage

```bash
python3 main.py
``` 

---
